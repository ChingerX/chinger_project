Адаптивные изображения

Демонстрация современных техник

1. Базовое адаптивное изображение

Что происходит:

  • Изображение занимает 100% ширины контейнера
  • Высота автоматически рассчитывается для сохранения пропорций
  • Работает на всех устройствах
  • Используется srcset для загрузки оптимального размера

2. Picture element с разными форматами

Что происходит:

  • Браузер выбирает WebP если поддерживает, иначе JPEG
  • Автоматическая оптимизация формата
  • Меньший размер файла при сохранении качества

3. Разные изображения для разных экранов

Что происходит:

  • На десктопе (≥1200px) загружается широкое изображение
  • На планшете (≥768px) загружается среднее изображение
  • На мобильном загружается портретное изображение
  • Экономия трафика и улучшение производительности

4. Retina display поддержка

Что происходит:

  • На обычных экранах загружается 1x версия
  • На Retina-экранах автоматически загружается @2x версия
  • Четкое отображение на всех устройствах

5. Object-fit свойства

object-fit: cover
Заполняет с обрезкой

object-fit: contain
Вписывает полностью

object-fit: fill
Растягивает

Что происходит:

  • cover: Изображение заполняет контейнер, обрезая лишнее
  • contain: Изображение вписывается полностью, сохраняя пропорции
  • fill: Изображение растягивается, игнорируя пропорции

6. Адаптивная галерея

Что происходит:

  • На десктопе: 4 колонки
  • На планшете: 2 колонки
  • На мобильном: 1 колонка
  • Автоматическое перестроение сетки при изменении размера экрана

7. Адаптивные формы

Что происходит:

  • Поля формы адаптируются под ширину экрана
  • На мобильных устройствах поля занимают 100% ширины
  • На десктопе поля имеют оптимальную ширину

8. Адаптивные таблицы

Проект Технологии Статус Дата
Личный сайт HTML, CSS, Bootstrap Завершен 2025-01-15
Todo приложение JavaScript, LocalStorage В разработке 2025-01-20
Интернет-магазин React, API Завершен 2025-01-25

Что происходит:

  • На десктопе: таблица отображается полностью
  • На мобильных: таблица становится карточками
  • Горизонтальный скролл на средних экранах
