# Портфолио студента - Чиркин Александр

Личный веб-сайт студента-разработчика. Проект представляет собой многостраничное портфолио, которое сочетает в себе функции визитной карточки, галереи проектов и учебного дневника.

## Структура проекта

```
base-project/
├── index.html              # Главная страница (визитная карточка)
├── pages/                  # Директория дополнительных страниц
│   ├── projects.html       # Страница проектов
│   ├── diary.html          # Учебный дневник
│   └── contacts.html       # Страница контактов
├── styles/                 # Директория стилей
│   └── styles.css          # Основной файл стилей
├── images/                 # Директория изображений
│   └── icons/              # Поддиректория для иконок
├── scripts/                # Директория скриптов (для КР3+)
├── assets/                 # Дополнительные ресурсы
│   └── favicon.ico         # Иконка сайта
└── README.md               # Документация проекта
```

## Описание страниц

### Главная страница (index.html)
Главная страница служит визитной карточкой студента. Включает:
- Информацию о студенте
- Навыки и технологии
- Проекты

### Страница проектов (projects.html)
На странице проектов собрана полная коллекция учебных и личных работ в виде адаптивной сетки. При клике на карточку проекта открывается модальное окно с детальной информацией: полным описанием, скриншотами, ссылками на живую версию и исходный код.

### Учебный дневник (diary.html)
Учебный дневник предназначен для отслеживания образовательного прогресса. Отдельная секция посвящена текущим курсам и предметам, где для каждого отображается прогресс в виде заполняющихся прогресс-баров.

### Страница контактов (contacts.html)
Основной элемент страницы - контактная форма с полями для имени, email адреса и текста сообщения. Форма включает валидацию вводимых данных для обеспечения корректности информации.
