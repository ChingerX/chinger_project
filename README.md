# Мой первый проект 
## Проект 
Этот репозиторий является каркасом для будущего веб-приложения. 

## Функциональность
- Многостраничная структура с семантической разметкой
- Форма обратной связи в модальном окне `<dialog>`
- Валидация формы с использованием HTML5 и JavaScript
- Клавиатурная доступность (Tab, Shift+Tab, Esc)
- Автоматическая маска для телефона
- Адаптивный дизайн с современными стилями
