# Pull Request: Практическое занятие №15 - Комбинирование методов адаптивной вёрстки

## Описание изменений

Реализованы все требования практического занятия №15 по комбинированию методов адаптивной вёрстки.

## Выполненные задачи

### ✅ 0) Preflight: Корректный viewport и каркас страницы
- Добавлен viewport meta tag с `viewport-fit=cover`
- Настроен базовый каркас с единицами высоты (vh/svh/dvh)
- Добавлен `color-scheme: light dark`
- Использована flexbox-структура для `.page`

### ✅ 1) Семантика + «ссылка-пропуск» + видимый фокус
- Добавлен skip-link для клавиатурной навигации
- Настроен видимый фокус с `:focus-visible`
- Добавлены крупные цели для touch-устройств (44px минимум)

### ✅ 2) Sticky-хедер и безопасные области (вырезы)
- Навигация с `position: sticky`
- Учет safe-area через `env(safe-area-inset-top)` и `env(safe-area-inset-bottom)`
- Компенсация вырезов на iOS

### ✅ 3) «Жидкий» фундамент (fluid foundation)
- Использован `clamp()` для адаптивных размеров
- Полоса во всю ширину с `calc(50% - 50vw)`
- Плавная типографика с `clamp()` для заголовков
- Адаптивные отступы с `clamp()`

### ✅ 4) Медиазапросы (MQ)
- Breakpoints для мобильных, планшетов и десктопов
- Адаптивные стили для разных размеров экранов

### ✅ 5) Flex/Grid для макета
- Grid для карточек проектов (`repeat(auto-fit, minmax())`)
- Flexbox для профиля и навигации
- Grid с явным «жёлобом» для двух колонок

### ✅ 6) Responsive images
- Адаптивные изображения с `srcset` и `sizes`
- Lazy loading для оптимизации
- Правильные аспектные соотношения

### ✅ 7) Контейнерные запросы (CQ)
- Использованы container queries для виджетов
- Адаптивные размеры шрифта через `cqi`
- Условная раскладка на основе размера контейнера

### ✅ 8) calc() для различных задач
- Кнопки с минимальной высотой 44px (WCAG)
- Смешивание % и px для адаптивности
- Компенсация safe areas
- Grid-колонки с явным «жёлобом»

### ✅ 9) Исправление функциональности карточек проектов
- Добавлены модальные окна с использованием нативного HTML5 `<dialog>`
- Восстановлена кликабельность карточек проектов
- Добавлен JavaScript для управления модальными окнами
- Стили для модальных окон с адаптивностью

## Изменённые файлы

- `index.html` - обновлена структура с использованием современных техник адаптивной вёрстки
- `styles/styles.css` - создан новый файл стилей с полной реализацией всех требований
- Добавлены модальные окна для проектов

- JavaScript (управление модальными окнами)
- Адаптивная вёрстка (clamp(), calc(), медиазапросы)
- Доступность (a11y) - skip-link, focus-visible, ARIA
