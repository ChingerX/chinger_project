# Pull Request: Практическое занятие №16 - Отзывчивое использование изображений

## Описание изменений

Реализованы все требования практического занятия №16 по отзывчивому использованию изображений, адаптивности форм, таблиц и масштабированию элементов.

## Выполненные задачи

### ✅ 1. Базовое адаптивное изображение
- Использование `srcset` и `sizes` для оптимизации загрузки
- Автоматическое масштабирование изображений
- Сохранение пропорций при изменении размера

### ✅ 2. Picture element с разными форматами
- Поддержка WebP с fallback на JPEG
- Автоматический выбор оптимального формата браузером
- Уменьшение размера файлов при сохранении качества

### ✅ 3. Разные изображения для разных экранов
- Медиа-запросы в `<picture>` элементе
- Оптимизированные версии для десктопа (≥1200px), планшета (≥768px) и мобильного
- Экономия трафика и улучшение производительности

### ✅ 4. Retina display поддержка
- Использование `srcset` с `1x` и `2x` версиями
- Четкое отображение на Retina-экранах
- Автоматическая загрузка высокого разрешения

### ✅ 5. Object-fit свойства
- Демонстрация `cover`, `contain`, `fill`
- Контроль заполнения контейнера изображением
- Визуальная демонстрация каждого варианта

### ✅ 6. Адаптивная галерея
- Grid с адаптивными колонками (4 → 2 → 1)
- Автоматическое перестроение при изменении размера экрана
- Hover-эффекты и lazy loading

### ✅ 7. Адаптивные формы
- Адаптивные поля ввода под ширину экрана
- Улучшенные фокус-стили и валидация
- Оптимальная ширина на разных устройствах

### ✅ 8. Адаптивные таблицы
- На мобильных устройствах превращаются в карточки
- Горизонтальный скролл на средних экранах
- Использование `data-label` для мобильной версии

### ✅ 9. Обновление существующих страниц
- Все изображения на `pages/projects.html` обновлены с `srcset`, `sizes` и `loading="lazy"`
- Форма на `pages/contacts.html` сделана адаптивной
- Навигация обновлена на всех страницах

## Изменённые файлы

- `pages/responsive-images-demo.html` - новая демо-страница с примерами всех техник
- `styles/responsive-images.css` - новый файл стилей для адаптивных изображений
- `pages/projects.html` - обновлены все изображения с адаптивными атрибутами
- `pages/contacts.html` - улучшена адаптивность формы
- `pages/diary.html` - обновлена навигация
- `index.html` - добавлена ссылка на демо-страницу в навигацию

## Технологии

- HTML5 (`<picture>`, `srcset`, `sizes`)
- CSS3 (Grid, Flexbox, Media Queries)
- Адаптивная вёрстка (responsive images, forms, tables)
- Lazy loading для оптимизации
- Поддержка темной темы (`prefers-color-scheme`)
- Адаптивность для печати

## Особенности реализации

- **Lazy loading**: Все изображения используют `loading="lazy"` для оптимизации
- **Темная тема**: Автоматическое переключение через `prefers-color-scheme`
- **Печать**: Оптимизированные стили для печати
- **Доступность**: Правильные alt-атрибуты и семантическая разметка
- **Производительность**: Оптимизация загрузки изображений через srcset
